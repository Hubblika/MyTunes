# MyTunes - music streaming app

## Requirements
git version 2.51.0 or above
Docker desktop with a docker version of 28.3.3 or above

## Installation
To get this project running locally, follow these steps:

1. Clone the repo:
   `git clone https://github.com/Hubblika/MyTunes.git`
2. Navigate into the project folder
    `cd MyTunes`
3. Pull the large files from the repo
    `git lfs pull`
4. Change to working directory
    `cd www`
5. Start docker desktop:
    `Open it on your machine`
6. Build and run the docker container:
    `docker-compose up --build`

## 📖 Usage
After starting the app, navigate to `localhost:8000`. You can start using the app.
The forgot password emails are avaliable on `localhost:8025`.
